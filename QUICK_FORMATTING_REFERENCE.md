## Quick Syntax Cheat Sheet

| Format | Syntax | Example |
|--------|--------|---------|
| **Bold** | `**text**` | Use **100% cotton** |
| Ordered List | `1. Item`<br/>`2. Item` | 1. Wash cold<br/>2. Air dry |
| Bullet List | `• Item` | • Cotton<br/>• Sustainable |
| Paragraph | Press Enter ×2 | Blank line between paragraphs |
| New Line | Enter (once) | Line within same paragraph |

---

## Common Mistakes to Avoid

❌ **Wrong:**
- `*bold*` (single asterisks - doesn't work)
- `**bold (missing closing)
- `1.Item` (no space after number)
- `•Item` (no space after bullet)

✅ **Right:**
- `**bold**` (double asterisks)
- `**bold**` (both sides)
- `1. Item` (space after number)
- `• Item` (space after bullet)

## Admin Panel Quick Buttons

You'll see 4 buttons under the description field:

1. **Bold Selected Text** 
   - Highlight text first
   - Click button
   - Text becomes `**text**`

2. **Insert Ordered List**
   - Click button
   - Gets 3 ready-to-edit items:
     ```
     1. First item
     2. Second item
     3. Third item
     ```

3. **Insert Bullet List**
   - Click button
   - Gets 3 ready-to-edit items:
     ```
     • Feature 1
     • Feature 2
     • Feature 3
     ```

4. **Add Care Instructions**
   - Click button
   - Gets template:
     ```
     **Care Instructions:**
     • Machine wash cold
     • Do not bleach
     • Lay flat to dry
     ```

## Pro Tips

💡 **Tip 1: Use Bold for Important Info**
- Material: `**100% organic cotton**`
- Price note: `**On Sale - 30% off**`

💡 **Tip 2: Lists for Readability**
- Features → Bullet list
- Instructions → Ordered list
- Mix them together!

💡 **Tip 3: Complete Example Template**
```
[One sentence hook - main selling point]

**Style Details:**
• Feature 1
• Feature 2
• Feature 3

**Care & Maintenance:**
1. First instruction
2. Second instruction
3. Third instruction

**Material:** [Details here]
**Colors:** [List colors]
**Sizing:** True to size / See chart
```

💡 **Tip 4: Preview Before Publishing**
1. Save as Draft
2. View product on website
3. Check formatting
4. Publish when ready
